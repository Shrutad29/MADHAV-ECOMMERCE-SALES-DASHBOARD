# MADHAV-ECOMMERCE-SALES-DASHBOARD
📊 Project Title / Headline
MADHAV ECOMMERCE SALES DASHBOARD
An interactive and analytical Microsoft Power BI Desktop dashboard built to explore e-commerce sales performance, customer behavior, profit trends, and product category insights across different states and time periods.

#  Short Description / Purpose

The MADHAV Ecommerce Sales Dashboard is a dynamic and visually interactive Power BI report designed to analyze online sales data and uncover meaningful business insights. The dashboard helps track sales, profit, quantity sold, customer purchasing behavior, and payment preferences through powerful visualizations and KPI metrics.

This dashboard is useful for business owners, sales analysts, e-commerce managers, and data analysts who want to monitor performance, identify trends, and make data-driven business decisions.


#  Tech Stack

The dashboard was built using the following tools and technologies:

* 📊 Microsoft Power BI Desktop – Main platform used for creating interactive dashboards and visual reports.
* ⚙️ Power Query – Used for data cleaning, transformation, and preprocessing.
* 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, and dynamic analysis.
* 🗂️ Data Modeling – Relationships created among tables for filtering and cross-analysis.
* 📁 Excel / CSV Dataset – Used as the primary source for e-commerce sales data.
* 🖼️ PNG Format – Used for dashboard preview and GitHub presentation.

# Features / Highlights

## • Business Problem

E-commerce businesses generate large amounts of sales data daily, making it difficult to manually track performance, customer behavior, and profitability trends.

Important business questions such as:

* Which states generate the highest sales?
* Which product category is most profitable?
* What payment method is most preferred?
* Which months show profit growth or loss?
* Who are the top customers?

…are difficult to analyze quickly using raw spreadsheets.


## • Goal of the Dashboard

The goal of this dashboard is to provide an interactive visual analytics solution that:

* Helps monitor overall sales and profit performance
* Tracks customer purchasing trends
* Identifies top-performing states and product categories
* Analyzes payment method usage
* Supports data-driven business decision making
* Simplifies complex e-commerce data into easy-to-understand visuals

#  Walkthrough of Key Visuals

## 🔹 KPI Cards (Top Section)

Displays key business metrics including:

* Total Sales Amount
* Total Profit
* Total Quantity Sold
* Average Order Value (AOV)

These KPIs provide a quick overview of overall business performance.

## 🔹 Sales by State (Bar Chart)

Shows state-wise sales distribution to identify high-performing and low-performing regions.


## 🔹 Quantity by Category (Donut Chart)

Displays contribution of categories such as:

* Clothing
* Electronics
* Furniture

Helps understand which categories sell the most products.


## 🔹 Profit by Month (Column Chart)

Visualizes monthly profit trends and highlights months with high or low profitability.


## 🔹 Payment Mode Analysis (Donut Chart)

Shows customer payment preferences including:

* COD
* UPI
* Debit Card
* Credit Card
* EMI

Helps businesses understand customer transaction behavior.


## 🔹 Profit by Sub-Category (Bar Chart)

Compares profits generated by different product sub-categories such as:

* Printers
* Bookcases
* Sarees
* Accessories

Identifies the most profitable products.


## 🔹 Customer Analysis

Displays top customers based on purchase amount and contribution to overall sales.


## 🔹 Interactive Filters / Slicers

Allows users to filter data dynamically by:

* Quarter
* State

Enabling customized analysis and deeper insights.
